========================
The django-admin command
========================

Moved to :doc:`/ref/django-cli`.

The ``django-admin`` command is being renamed to ``django``,
so this page has been moved to match the new name.
The ``django-admin`` command remains available,
and there is no deprecation timeline for removal.
For more details on the naming change, see `DEP 16`_.

.. _`DEP 16`: https://github.com/django/deps/blob/main/final/0016-name-the-main-command-django.rst
