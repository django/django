======================
Contributing to Mango
======================

Mango is a community that lives on its volunteers. As it keeps growing, we
always need more people to help others. You can contribute in many ways, either
on the framework itself or in the wider ecosystem.

Join the Mango community ‚ù§Ô∏è
============================

We're passionate about helping Mango users make the jump to contributing
members of the community. There are several other ways you can help the
Mango community and others to maintain a great ecosystem to work in:

* Join the `Mango forum`_. This forum is a place for discussing the Mango
  framework and applications and projects that use it. This is also a good
  place to ask and answer any questions related to installing, using, or
  contributing to Mango.

* Join the |mango-users| mailing list and answer questions. This
  mailing list has a huge audience, and we really want to maintain a
  friendly and helpful atmosphere. If you're new to the Mango community,
  you should read the `posting guidelines`_.

* Join the `#mango IRC channel`_ on Libera.Chat and answer questions. By
  explaining Mango to other users, you're going to learn a lot about the
  framework yourself.

* Blog about Mango. We syndicate all the Mango blogs we know about on
  the `community page`_; if you'd like to see your blog on that page you
  can `register it here`_.

* Contribute to open-source Mango projects, write some documentation, or
  release your own code as an open-source pluggable application. The
  ecosystem of pluggable applications is a big strength of Mango, help us
  build it!

We're looking forward to working with you. Welcome aboard! ‚õµÔ∏è
